# 🤖 Yieldra: Lend your money for AI to work 

**Yieldra** is an industrial-grade "Liquidity-as-a-Service" (LaaS) protocol built on the **Arc Network**. It enables a new economic paradigm: **H2A (Human-to-Agent)**. Finally, humans can lend their idle capital directly to autonomous AI agents to execute high-frequency on-chain strategies.

---

## 🌟 The H2A Economy (Human-to-Agent)

Yieldra eliminates the friction of traditional collateralized lending by introducing the **Pure Trust** model:
- **Lenders (Humans):** Put your money to work! Deposit USDC into secure vaults and earn yield generated by machine intelligence.
- **Borrowers (Agents):** Verified autonomous agents utilize this liquidity for sub-second arbitrage, market making, and liquidations.
- **Yield Source:** Real economic revenue from high-frequency transactions and algorithmic interest rates.

## ✨ Core Innovations

### 🛡️ Pure Trust Credit (ERC-8004)
Traditional DeFi requires 150%+ collateral. Yieldra enables **0% collateral credit** for verified agents based on their on-chain Reputation Score. This allows machines to scale capital at the speed of computation.

### ⚡ Nanopayments at Scale (The Circle Integration)
Yield isn't a "black box." Yieldra handles transactions that are impossible on other chains:
- **Sub-Cent Settlement:** Agents settle actions for as low as **$0.0001**.
- **Arc Native USDC:** Settlement occurs in native USDC with sub-second finality.
- **Economic Viability:** Zero gas overhead means 100% of the micro-fees go toward your yield.

## 🦾 Running the Autonomous Demo (Turbo Mode)

For the ultimate hackathon presentation, you can run our updated **Turbo Agent Bot** that programmatically generates constant on-chain revenue while you present.

### 🚀 Launching the Agent Bot
1. Ensure you have your `PRIVATE_KEY` set in `contracts/.env`.
2. Open a terminal and navigate to the `contracts` folder.
3. Run the following command:
   ```bash
   npx hardhat run scripts/autonomous_agent.ts --network arcTestnet
   ```
4. **The Result**: The bot will start injecting realistic nanopayment fees ($0.0001 - $0.30) every **10-30 seconds**. Your dashboard will pulse in real-time showing the "Agentic Economy" in action.

### Installation
```bash
# Clone and install
git clone https://github.com/tetheusz/yieldra.git
npm install

# Start the Engine
npm run dev
```
